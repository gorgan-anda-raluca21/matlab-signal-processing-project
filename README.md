# MATLAB Signal Processing & System Identification Project

## 📌 Description

This project focuses on system identification and signal processing using MATLAB and Simulink. It includes signal generation, modeling, and analysis of dynamic systems such as electrical circuits and pendulum systems.

The project combines theoretical concepts with practical implementation, supported by simulations and experimental analysis.

---

## ⚙️ Technologies Used

* MATLAB
* Simulink
* Signal Processing techniques
* System Identification methods

---

## 🧠 Project Structure

### 🔹 MATLAB Scripts (`/src`)

* Signal generation (chirp signals, input signals)
* Linear regression modeling
* Dynamic system modeling (pendulum system)

### 🔹 Simulink Models (`/simulink`)

* Electrical circuit simulations
* Pendulum system simulation
* System validation using block-based modeling

### 🔹 Documentation (`/docs`)

* Detailed project report including theory, implementation, simulations, and results

---

## 🔬 Methodology

1. **Signal Generation**

   * Creation of input signals for system analysis

2. **System Modeling**

   * Mathematical modeling of dynamic systems
   * Implementation using MATLAB scripts

3. **Simulation (Simulink)**

   * Modeling systems using block diagrams
   * Validation of system behavior

4. **Data Analysis**

   * Applying regression techniques
   * Evaluating system responses

---

## 📊 Results

* Simulation of dynamic systems (electrical circuits, pendulum)
* Analysis of system responses to different inputs
* Validation of theoretical models through Simulink
